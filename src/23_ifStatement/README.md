# If Statement 
* Sama seperti di JavaScript, di TypeScript kita juga bisa melakukan if statement
* If statement di TypeScript sama seperti di JavaScript.
* Kode: If Statement
  ```TSX
  const examValue = 90;

  if (examValue > 80) {
    console.info('Good');
  } else if (examValue > 60) {
    console.info('Not Bad');
  } else {
    console.info('Try Again');
  }
  ```
